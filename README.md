# messenger-bot
A messenger bot written in go

Trying to build a messenger bot.

It's written in go and deployed to Heroku.

Try it.
Clone it.
heroku create
git push heroku master
After that you will need to add variables:
heroku config:set VALIDATION_TOKEN=********
